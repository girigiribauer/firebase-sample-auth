# sample-firebase-auth

see <https://firebase.google.com/docs/functions/config-env>

## 1. npm install

    npm install

## 2. edit config files

* config/apikey.env.js
* .firebaserc

## 3. configure firebase console

Authentication > provider

* Mail / Password
* Phone
* Google
* Facebook
* Twitter
* GitHub
* Anonymous

## 4. npm start

    npm start
